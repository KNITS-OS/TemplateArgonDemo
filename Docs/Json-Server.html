Tutorial

Step 1 Install json-server as dev dependency

Type in terminal

>  npm install -D json-server

Step 2 add script to package.json

In ./package.json

...
"scripts": {
	"start": "react-scripts start",
	"build": "react-scripts build",
	...
	"json-server": "json-server --watch ./json-server-mock-data/db.js"
}
...

Step 3 Create db.js file

In ./json-server-mock-data/db.js

// mock data files to import
const employees = require("./employees.js");
const businessUnits = require("./business-units.js");
const countries = require("./countries.js");
const groups = require("./groups.js");

module.exports = () => ({
	employees,
	businessUnits,
	countries,
	groups
});

Step 5 start json-server

Type in terminal

>  npm run json-server

Step 6 verify api on json-server

With curl from Linux Bash terminal

>  curl http://localhost:3000/employees

http://localhost:3000/employees
